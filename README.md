# Vue3后台管理系统（已上线）
## author：QC
项目概述：此项目为一个微型后台管理系统，具备基础的后台管理功能，如权限管理、角色管理等。后台管理系统是一个非常通用的系统，任何复杂的后台管理系统都可以基于此系统来做进一步的开发
